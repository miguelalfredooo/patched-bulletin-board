# Patched Framework

The Patched framework is a modular philosophy for designing AI-driven experience systems. It's a shared vocabulary for designers, engineers, and product managers.

## What's Here

- **[PATCHED.md](PATCHED.md)** — Complete framework documentation
- **[PATCHED-EXAMPLE.md](PATCHED-EXAMPLE.md)** — Worked example applying the framework

## Quick Overview

Patched structures systems using:
- **Module layer** — Reusable components and tools
- **Behavior layer** — Agent actions and decision logic
- **Governance layer** — Rules, constraints, approval flows
- **Signal flow** — How information moves through the system

## Using This Framework

The Patched framework helps with:
- Designing systems with AI agents
- Thinking modularly about complex workflows
- Creating repeatable patterns
- Scaling teams and systems

This project (**Design By Bulletin**) applies Patched principles to the editorial publication system.
