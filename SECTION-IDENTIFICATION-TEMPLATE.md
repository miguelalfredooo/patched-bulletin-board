# Section Identification Template — Quick Reference

Every ASCII section needs clear identification showing **section number** and **section name/theme**. Choose the format based on your compositional approach (Bauhaus Grid, Composition, or Typography as Design).

---

## Template A: Bauhaus Grid Approach

Use consistent, heavy headers that align to the grid. Priority on structure and modularity.

### Primary Section Header (before ASCII art):
```
═══════════════════════════════════════════════════════════════
  SECTION 1 — Art (Format A: Classic Object)
═══════════════════════════════════════════════════════════════
```

### Secondary Header (mid-section, for narrative):
```
───────────────────────────────────────────────────────────────
  The First Impression
───────────────────────────────────────────────────────────────
```

### Minimal Marker (for simple sections):
```
 ▸ SECTION 2 — Painting
```

**Pattern:** Consistent line weight signals consistent approach. Reader learns rhythm immediately.

---

## Template B: Composition Approach

Use headers that vary by section weight/density. Priority on pacing and rhythm.

### Heavy Section (before large ASCII):
```
╔════════════════════════════════════════╗
║  SECTION 1 — Opening (Full Weight)    ║
╚════════════════════════════════════════╝
```

### Medium Section (before moderate ASCII):
```
  ┌────────────────────────────────────┐
  │  SECTION 3 — Building (Medium)     │
  └────────────────────────────────────┘
```

### Light Section (before sparse ASCII or breathing room):
```
       ▪ SECTION 6 — Pause ▪
```

**Pattern:** Header weight matches section weight. Reader feels the composition.

---

## Template C: Typography as Design Approach

Use thematic headers that are statements, not just labels. Priority on narrative voice.

### Statement Header (narrative-driven):
```
═══════════════════════════════════════════════════════════════
  SECTION 1 — THE VOID SPEAKS
═══════════════════════════════════════════════════════════════
```

### Bridge Header (connecting ideas):
```
───────────────────────────────────────────────────────────────
  What does silence sound like?
───────────────────────────────────────────────────────────────
```

### Reflective Header (minimal, poetic):
```
       Why does order emerge from chaos?
```

**Pattern:** Headers tell part of the story. Readers engage with text as primary.

## Naming Convention

### Section Name Format

**For all three approaches, use:**
```
SECTION [#] — [Name/Theme]
```

Where [#] is 1-11 and [Name/Theme] describes the section's role in the issue.

#### Examples:

**Bauhaus Grid:**
```
SECTION 1 — Discovery (Format A)
SECTION 2 — Recognition (Format B)
```

**Composition:**
```
SECTION 1 — Opening (Dense)
SECTION 6 — Breathing Room
```

**Typography:**
```
SECTION 1 — THE VOID SPEAKS
SECTION 4 — WHERE MEANING CRYSTALLIZES
```

## Line Weight Choices

| Line Weight | Meaning | Use In |
|------------|---------|---------|
| `═══════` | Heavy | Primary sections, visual peaks, major statements |
| `───────` | Light | Secondary ideas, transitions, breathing |
| `╭─────╮` | Medium | Contained thoughts, modular sections |
| `░░░░░░` | Texture | Backgrounds, density variation |
| `◆` | Minimal | Pauses, silence, reset points |

---

## Quick Decision Tree

**When creating a new section, ask:**

1. **Is this primary content?** 
   - Yes → Heavy header (═══)
   - No → Light header (───)

2. **Is this a narrative shift?**
   - Yes → Bridge header (light, transitional)
   - No → Continue with consistent weight

3. **Is this a pause or breath?**
   - Yes → Minimal marker (◆ or simple text)
   - No → Full header with section name

4. **Does the header need to be a statement?**
   - Yes (Typography approach) → Thematic language ("THE VOID SPEAKS")
   - No (Grid/Composition) → Functional language ("Discovery" or "Format A")

---

## Final Checklist: Every Section Should Have

- [ ] Section number clearly identified (SECTION 1, 2, 3, etc.)
- [ ] Section name/theme visible (Art, Discovery, Emergence, etc.)
- [ ] Header weight appropriate to content
- [ ] Header text serves the approach (structural, paced, narrative)
- [ ] Breathing room after header before ASCII art
- [ ] Breathing room after ASCII art before next section
- [ ] No two consecutive sections have identical header weight

Apply this template consistently across all 11 sections. The reader will learn the visual language and trust the structure.
