COMPOSITION NOTES

These letter blocks demonstrate:

**Heavy Style (Style 1):**
- Dense fills (█)
- Strong presence on page
- Best for headlines, emphasis
- High visual weight

**Bordered Style (Styles 2-3):**
- Clear containment
- Modular, stackable
- Medium weight
- Professional, structured

**Minimal Style (Styles 4-5):**
- Sparse linework
- Elegant, refined
- Negative space design
- Lower visual weight

**Textured Style (Style 8):**
- Gradient fills (░▒▓█)
- Depth through density
- Dynamic, living quality
- Medium-to-heavy weight

**Individual Showcases:**
Each letter is a composition unto itself. A = triangular peak
(pyramid hierarchy). M = multiple peaks (rhythm, repetition).
O = containment (positive/negative balance). S = curvature
(tension, flow).

Use letter blocks to:
- Create typographic emphasis
- Build visual hierarchy
- Vary page weight (heavy letters anchor)
- Express semantic meaning (shape = meaning)
- Layer letters into words/phrases
