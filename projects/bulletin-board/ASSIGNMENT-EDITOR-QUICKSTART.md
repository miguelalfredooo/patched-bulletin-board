## ASSIGNMENT WORKFLOW

### Step 1: Pick Your Issue
Select from Issues 003-010 above. Cross-reference archive-log.md for the full metadata.

### Step 2: Set Editorial Mix
Copy the default mix from the Quick Reference, or adjust based on briefing from Editorial Director. The mix modulates:
- **Music 0%** → prose ignores sonic qualities
- **Music 100%** → every sentence mirrors tempo, rhythm, production texture
- **Visual 0%** → ASCII is decorative; prose drives structure
- **Visual 100%** → ASCII composition determines section order, prose follows visual momentum
- **Research 0%** → surface-level curation only
- **Research 100%** → deep signal synthesis across sources, 4-5 convergence layers
- **Process 0%** → maker perspective absent
- **Process 100%** → "how is this made?" centers every selection
- **Theme 0%** → invisible, felt only through aesthetic coherence
- **Theme 100%** → explicit, woven into every sentence
- **AI Culture 0%** → tools and AI absent
- **AI Culture 100%** → AI/tool philosophy featured in every section

### Step 3: Scout & Curate
Use the **Prose Brief** from CONTENT-BRIEFS-003-010.md for each section. This guides your research angle.

### Step 4: Verify Format Sequence
Check that no two consecutive sections use the same Format (A-E). Vary visual weight naturally.

### Step 5: Confirm Source Diversity
Track every source used. No repeats except music platforms (Bandcamp, Discogs, Spotify, YouTube can repeat).

### Step 6: Write Prose
For each section, write 1-2 sentences (usually 2) following the brief angle. The prose should:
- Be specific to this piece, this day, this theme
- Not explain the theme (it should be felt, not stated)
- Reflect the Editorial Mix intensity
- Use rhythm and word choice informed by the Sonic Reference if Music% is high
- Support visual momentum if Visual% is high

### Step 7: Final Polish
- **Closing Sentence:** Should land with thematic weight without being heavy-handed
- **Cultural Thread:** Explain the invisible connection across sources (3-4 sentences, no audience)
- **Curator Note:** Reflect on decisions and signal distinctions from adjacent issues

## PROSE DEVELOPMENT CHECKLIST

- [ ] Each section prose is specific to the curated work, not generic to theme
- [ ] No two consecutive formats are identical
- [ ] All 11 sections use different sources (except music)
- [ ] Theme is felt, not stated (check: can you remove the word "theme" entirely and still feel it?)
- [ ] Prose rhythm varies across sections (don't let every sentence follow the same cadence)
- [ ] Visual weight matches Format assignment (A = light, E = heavy)
- [ ] Closing Sentence lands without being preachy
- [ ] Cultural Thread connects sources without over-explaining
- [ ] Curator Note acknowledges what this issue is *not* (distinction from adjacent themes)

## COMMON PITFALLS

**Prose that explains the theme:**
❌ "This artist embraces handmade values by rejecting digital tools."
✅ "The split nib line is unrepeatable; no two letters match even in the same sentence."

**Source repeats:**
❌ Using Artsy twice (once for Art, once for Painting)
✅ Track sources as you assign; swap in alternative publications if you repeat

**Formulas that feel generic:**
❌ "This designer is exploring the intersection of [theme] and [medium]."
✅ "[Artist name]'s [specific work] [specific gesture/choice that exemplifies the theme implicitly]."

**Visual weight mismatches:**
❌ Format A (sparse, light) paired with heavy prose sentences
✅ Keep prose rhythm light for A, dense for E

**Theme overstatement:**
❌ Every section explicitly mentions the theme word
✅ The theme word should appear only in the Closing Sentence and Brief headers
