**The Logic of Scatter: Algorithms and Emergence**

Scatter isn't random anymore. It's algorithmic. Machine learning systems can distribute elements with apparent randomness that actually follows complex rules. The result looks scattered but is secretly structured—a hidden order beneath the scatter.

Cities grow in scatter patterns. Coral reefs form scattered branches. Constellations are scattered but connected by invisible lines we draw. Nature's scatter always contains logic. Evolution discovered that distributed, scattered systems are more resilient than centralized ones.

Design is learning from this. Information architecture now uses scattered, distributed networks rather than hierarchical trees. UX patterns scatter elements across screens in ways that follow user behavior rather than designer intuition. The scatter is data-driven. The apparent randomness is actually optimization.
