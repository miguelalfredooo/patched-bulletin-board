**Design & AI Tools — Tool Decay and Obsolescence**

Software degrades. Interfaces collapse after updates, features disappear, entire tools become unrecognizable. But there's another kind of erosion: AI tools that get worse the more you use them, tending toward average output, consensus aesthetic, the flattened middle. The tool decays not through technical failure but through overuse. Every prompt sharpens the system's tendency toward mediocrity. The designer who built something singular with first-generation tools watches it become generic when the algorithm learns. Tool obsolescence isn't just about versioning. It's about how systems inevitably smooth themselves toward least resistance. The question for designers: do you resist this erosion or design around it?
