# Illustration

Illustration lives closest to the icon of any visual discipline. Susan Kare knew this when she drew the original Mac icons — the bomb, the trash can, the smiling finder — using a 32×32 grid and a binary palette. Each image had to be legible at the size of a thumbnail and carry a concept that needed no caption. That constraint is what illustration does at its best: distill. Christoph Niemann has spent a career asking how little line it takes to read a face, a city, an idea. The answer is always less than you think.
