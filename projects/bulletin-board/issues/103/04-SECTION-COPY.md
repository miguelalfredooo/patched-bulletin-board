**Scatter in Language: Broken Linearity**

We read left to right. Top to bottom. Linear. But now we write scattered: hashtags, links, mentions appearing anywhere. We read in jumps, following links rather than lines. The page is no longer a rectangle you traverse linearly. It's a network where you scatter across paths.

This is how knowledge actually works. Ideas are scattered. To understand, you jump between sources, between disciplines, between perspectives. Linear thinking is beautiful, but scattered thinking is honest. It admits that understanding is not a line. It's a navigation through scattered pieces.

Typography is learning this. Text doesn't need to be columnar. It can scatter across space and still be readable. In fact, scattered text sometimes reads faster because your eye doesn't get stuck on one line. Scatter respects how attention actually moves—not linearly, but in jumps and spirals.
