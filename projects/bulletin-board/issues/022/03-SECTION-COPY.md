# Critical Thinking

Design theory has spent a century learning what artists knew intuitively: constraints create power. Dieter Rams didn't invent "less is more"—he named what was already true.

The Bauhaus removed ornament not for asceticism but for clarity. Every line had to justify itself. If a shape could be removed without losing meaning, it was removed. This was philosophy, not style.

Today's design systems are built on the same principle. A grid is negative space made visible. Every pixel of white space is a choice. Modern design is ruthlessly subtractive because subtraction is the only path to essential.
