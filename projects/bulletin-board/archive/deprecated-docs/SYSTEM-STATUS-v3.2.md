## How to Use This System

**For Editorial Director:**
1. Use AGENT-BRIEF-TEMPLATE.md (v3.2) to create brief
2. Send to Managing Editor + Design
3. Wait for deliveries (~2 hours)
4. Use EDITORIAL-SPOT-CHECK-PROCEDURE.md to verify
5. Announce completion

**For Managing Editor:**
1. Receive brief from Editorial
2. Write 11 sections, save as `NN-SECTION-COPY.md`
3. Self-validate using checklist in brief
4. Report: copy complete + observations

**For Design Department:**
1. Receive brief from Editorial
2. Read available styles in SECTION-ART-STYLES-INDEX.md
3. Choose styles that fit each section theme
4. Create 12 files (cover + 11 sections + footer)
5. Self-validate using checklist in brief
6. Report: art complete + style observations

**For Bulletin Bot:**
1. Receive approved issue from Editorial
2. Load 24 files
3. Format for Telegram (code blocks for visuals, MarkdownV2 for copy)
4. Publish issue
