# Section Art Styles Index

**Master reference for all available section art design directions**

---

## Available Styles

### 1. Bold Fill Approach
**File:** `SECTION-ART-BOLD-FILL-STYLE.md`

**Three primary approaches:**
- **Geometric** — Nested, symmetrical, inward-focusing
- **Divided** — Left/right split with equal visual weight
- **Center Focus** — Nested center with radiating field

**Best for:** Visual weight, balance, confidence, bold shapes
**Visual language:** Solid fills (█) and light fills (░)

---

### 2. Bauhaus Approach
**File:** `SECTION-ART-BAUHAUS-STYLE.md`

**Ten primary approaches:**
- **Circle** — Unity, cycles, wholeness
- **Triangle** — Direction, movement, progression
- **Grid** — Systems, order, modularity
- **Typography as Form** — Language as visual design
- **Asymmetry** — Balanced imbalance, sophistication
- **Line** — Direction, rhythm, intervals
- **Primary Shapes** — All three forms unified
- **Rhythm** — Repetition, pattern, beat
- **Contrast** — Drama, opposition, binary clarity
- **Modular** — Scalability, systems, flexibility

**Best for:** Geometric clarity, functional beauty, systematic thinking
**Visual language:** Primary forms (●, ■, △), geometric construction, typography integration

## How to Use

1. **Read the style guide** for your chosen approach
2. **Review examples** in the guide
3. **Check checklist** before delivery
4. **Choose approach** that matches section theme
5. **Create visual** following style principles
6. **Validate** against checklist

## Integration with Briefs

Design Department receives AGENT-BRIEF-TEMPLATE.md which references:
- **Preferred style:** Bold Fill Approach
- **Optional style:** Bauhaus Approach
- **Format distribution:** A/B/C/D/E (5 formats across 11 sections)

Mix and match styles for variety across 11 sections. Not every section needs the same style — choose what fits the theme.

## Adding New Styles

When exploring new style directions:
1. Create 3-10 experimental mocks
2. Document principles and examples
3. Create `SECTION-ART-[STYLE-NAME]-STYLE.md`
4. Add entry to this index
5. Update AGENT-BRIEF-TEMPLATE.md if style becomes preferred
