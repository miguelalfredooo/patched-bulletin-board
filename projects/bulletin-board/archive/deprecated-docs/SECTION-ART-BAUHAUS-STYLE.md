# Section Art Style Guide — Bauhaus Approach

**Version:** 1.0  
**Date:** May 9, 2026  
**Status:** Active design direction for section art

---

## Overview

Bauhaus style brings geometric clarity, functional beauty, and systematic thinking to section art. Form follows function. Typography is design. Repetition creates rhythm.

**Core Principle:** Every element must serve a purpose. Geometric forms are the fundamental vocabulary.

---

## Ten Bauhaus Approaches

### 1. CIRCLE — Primary Form

**Best for:** Unity, cycles, wholeness, continuity, eternity

**Characteristics:**
- Circle as fundamental geometric form
- Arc construction using dots or simple characters
- Center point marked (●)
- Typography integrated as label
- Symmetrical, complete composition

**Example:**
```
         . · · · .                
      ·             ·             
    ·                 ·           
   ·                   ·          
   ·         ●         ·          
   ·                   ·          
    ·                 ·           
      ·             ·             
         · · · ·                  
```

**When to use:**
- Cycles, renewal, eternal return
- Unity, wholeness, completeness
- Pure geometric abstraction
- Meditation, centeredness, presence

---

### 2. TRIANGLE — Primary Form

**Best for:** Direction, movement, progression, energy, hierarchy

**Characteristics:**
- Triangle built line by line from apex to base
- Geometric progression creating depth
- Base filled with solid marks
- Typography integrated (labels or statement)
- Asymmetrical construction, centered result

**Example:**
```
                 ▲                
                ▲ ▲               
               ▲   ▲              
              ▲     ▲             
             ▲       ▲            
            ▲         ▲           
           ▲           ▲          
          ▲▲▲▲▲▲▲▲▲▲▲▲▲           
```

**When to use:**
- Direction, forward movement, progress
- Energy, dynamism, growth
- Hierarchy, pyramid structure
- Apex = focus, base = foundation

---

### 3. GRID — Modular Composition

**Best for:** Systems, order, organization, modularity, structure

**Characteristics:**
- Box-drawing characters for gridlines
- Consistent modules (cells) throughout
- Checkerboard or ordered fill pattern
- Repetition creates visual rhythm
- Functional and systematic

**Example:**
```
┌──┬──┬──┬──┬──┬──┬──┬──┐         
│  │██│  │██│  │██│  │██│         
├──┼──┼──┼──┼──┼──┼──┼──┤         
│██│  │██│  │██│  │██│  │         
├──┼──┼──┼──┼──┼──┼──┼──┤         
│  │██│  │██│  │██│  │██│         
└──┴──┴──┴──┴──┴──┴──┴──┘         
```

**When to use:**
- Systems thinking, organization
- Modularity, scalability
- Factory, construction, precision
- Data, information, structure
- Repetition as strength

---

### 4. TYPOGRAPHY AS FORM — Type as Design

**Best for:** Language, communication, letterform beauty, pixel art

**Characteristics:**
- Letters constructed from solid blocks (█)
- Bitmap/pixel style typography
- Typography becomes visual art, not just text
- Form and content merged
- Functional clarity

**Example:**
```
██████  ██████  ██  ██            
██  ██  ██      ██  ██            
██████  ████    ██  ██            
██  ██  ██      ██  ██            
██  ██  ██████  ██████            
```

**When to use:**
- Words, language, communication
- Typography as design element
- Pixel/digital aesthetic
- Clarity, legibility, precision
- Form = content

---

### 5. ASYMMETRY — Balanced Imbalance

**Best for:** Tension, dynamism, stability through imbalance, maturity

**Characteristics:**
- Uneven placement of forms
- Visual weight balanced despite asymmetry
- Off-axis but stable composition
- Mix of solid and line elements
- Intentional, functional imbalance

**Example:**
```
████████████████                  
████████████████                  
████████████████                  
                                  
                  ────────────────
                                  
                           ●      
```

**When to use:**
- Sophistication, maturity, adulthood
- Tension, conflict, resolution
- Dynamic stability
- Complex balance
- Modern aesthetic

### 6. LINE — Primary Element

**Best for:** Direction, rhythm, flow, intervals, tempo

**Characteristics:**
- Horizontal lines as primary design element
- Converging or parallel arrangement
- Deliberate spacing creates rhythm
- Intervals create visual beat
- Minimal but powerful

**Example:**
```
──────────────────────────────────
  ────────────────────────────    
    ──────────────────────        
      ──────────────────          
        ────────────              
          ────────                
```

**When to use:**
- Motion, flow, direction
- Rhythm, tempo, beat
- Intervals, spacing, breathing
- Minimalism, elegance
- Time, progression, tempo

### 7. PRIMARY SHAPES — All Three Forms

**Best for:** Completeness, democracy of forms, fundamental design language

**Characteristics:**
- Circle, square, triangle present together
- Each form given equal visual weight
- Typography integrating all forms
- Equation: all forms are design elements
- Harmony through variety

**Example:**
```
  ████████                        
  ████████    . · · .             
  ████████   ·       ·            
  ████████   ·   ●   ·            
  ████████    · · · ·             
           ▲                      
          ▲ ▲                     
         ▲▲▲▲▲                    
```

**When to use:**
- Fundamental design principles
- Teaching, learning, pedagogy
- Design language, vocabulary
- Harmony, balance, democracy
- All-encompassing themes

### 8. RHYTHM — Repetition & Intervals

**Best for:** Music, tempo, heartbeat, pattern, regularity

**Characteristics:**
- Repeating symbols (▲ █ ● ─ ░)
- Rows of identical repeated forms
- Spacing creates intervals
- Visual beat, like music
- Mechanical, systematic precision

**Example:**
```
▲  ▲  ▲  ▲  ▲  ▲  ▲  ▲  ▲  ▲  ▲   
█  █  █  █  █  █  █  █  █  █  █   
●  ●  ●  ●  ●  ●  ●  ●  ●  ●  ●   
─  ─  ─  ─  ─  ─  ─  ─  ─  ─  ─   
```

**When to use:**
- Rhythm, beat, tempo, music
- Pattern, repetition, cycles
- Heartbeat, pulse, vitality
- Industrial aesthetic
- Mechanical precision

### 9. CONTRAST — High Drama

**Best for:** Opposition, drama, clarity, binary thinking, maximum impact

**Characteristics:**
- 50/50 split (solid vs light)
- Maximum visual contrast (█ vs ░)
- Simple division (left/right or top/bottom)
- No middle ground, no compromise
- Powerful, striking, dramatic

**Example:**
```
████████████████░░░░░░░░░░░░░░░░░░
████████████████░░░░░░░░░░░░░░░░░░
████████████████░░░░░░░░░░░░░░░░░░
████████████████░░░░░░░░░░░░░░░░░░
```

**When to use:**
- Opposition, contrast, duality
- Drama, power, impact
- Black/white thinking (when appropriate)
- Maximum clarity, no ambiguity
- Binary choices

### 10. MODULAR — System Composition

**Best for:** Scalability, systems, variation within rules, flexibility

**Characteristics:**
- Repeating 2-row modules
- Checkerboard offset pattern
- Alternating solid (█) and light (░)
- Systematic variation
- Functional beauty through modularity

**Example:**
```
████  ░░░░  ████  ░░░░  ████  ░░  
████  ░░░░  ████  ░░░░  ████  ░░  
░░░░  ████  ░░░░  ████  ░░░░  ██  
░░░░  ████  ░░░░  ████  ░░░░  ██  
```

**When to use:**
- Systems thinking, scalability
- Repetition with variation
- Modules, components, building blocks
- Flexibility within rules
- Industrial design principles

## Bauhaus Checklist (Before Delivery)

- [ ] Geometric form is clear and unambiguous
- [ ] Typography (if used) enhances the message
- [ ] No ornamentation, only essence
- [ ] Functional purpose is evident
- [ ] Visual hierarchy is strong
- [ ] Every element serves a purpose
- [ ] Composition is balanced (symmetrical or intentionally asymmetrical)
- [ ] Every line = exactly 34 characters
- [ ] Max 15 lines total
- [ ] No placeholder text (visual only)

---

## When to Use Bauhaus Style

✅ **Ideal for:**
- Systematic, ordered concepts
- Geometric abstraction
- Precision, clarity, function
- Modernist aesthetic
- Educational content
- Systems and structure

❌ **Not ideal for:**
- Organic, flowing, gestural themes
- Nature, landscape, living systems
- Literal representation
- Chaos, complexity, disorder (unless intentional)
- Handmade, artisanal aesthetic

---

## Key Principles (Always)

1. **Form follows function** — Every shape has purpose
2. **Geometric clarity** — No ambiguity, no fuzz
3. **Systematic thinking** — Rules create beauty
4. **Typography as design** — Words are visual elements
5. **Reduction** — Remove everything unnecessary
6. **Modularity** — Build from simple units
7. **Rhythm through repetition** — Pattern creates meaning
8. **Functional beauty** — Aesthetics and purpose merged
