# Design By Bulletin™ — Data Flow Between Agents

**Question:** What data gets passed from one agent to the next?  
**Answer:** Files. Each agent reads files from previous agent(s), processes them, and writes new files for the next agent.

---

## Summary: Data Lineage

```
Web Search Results (40-60 pieces)
         ↓
Art Department Curation (14 pieces selected + annotated)
         ↓
Assignment Editor Commissions (6 briefs with sources)
         ↓
Managing Editor Articles (6 articles citing sources)
         ↓
Editorial Director Validation (verifies curation visible)
         ↓
Bot Assembly (renders 6-section magazine)
         ↓
Telegram Subscribers (see final magazine with all curation visible)
```

## Key Point: Curation is Visible Throughout

**Every file the subscriber sees contains:**
- ✅ Sources from web search
- ✅ Art Dept's curation choices
- ✅ Assignment Editor's section mapping
- ✅ Managing Editor's article writing
- ✅ Editorial Director's approval

**Section 6 specifically shows:**
- 4 curated pieces on craft/objects/making
- Each cited in the article
- Art Dept's editorial insight about material culture
- Managing Editor's synthesis of curated sources
- Reader learns "why these 4 pieces matter together"

*Data flows one direction: Search → Curation → Commission → Writing → Validation → Publishing*

*Each stage adds value. Nothing is lost. All sources are credited.*
