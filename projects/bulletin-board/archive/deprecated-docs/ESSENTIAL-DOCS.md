## 🚀 Getting Started Fast

### **For Editorial Director (Create Issue 021 Now)**
```
1. Read: COMPLETE-SYSTEM-GUIDE.md (#editorial-director-quick-start) — 5 min
2. Copy: EDITORIAL-BRIEF-TEMPLATE.md — Fill in your values — 15 min
3. Send: Brief to agents — Instant
4. Wait: For Art Dept, Assignment, Managing to complete
5. Validate: python3 validator.py issues/021 — 2 min
6. Approve: python checkpoint.py signal editorial 021 "Validation passed..." — 1 min
7. Done: Bot publishes at 8 PM

Total work: ~30 minutes spread over 2 days
```

### **For Art Department (Curate Issue 021)**
```
1. Read: Brief from Editorial Director — 10 min
2. Run: python art-department-search.py --theme "[theme]" — 2 min
3. Curate: Evaluate 40-60 pieces, select 14-16 — 45 min
4. Report: Create art-department-report.md — 15 min
5. Signal: python checkpoint.py signal art-department 021 "14 pieces curated" — 1 min
6. Done: Hand off to Assignment Editor

Total work: ~1.5 hours (8:00-9:15 AM)
```

### **For Assignment Editor (Commission Issue 021)**
```
1. Read: Art Dept curation report — 10 min
2. Create: 6 commission briefs with sources — 45 min
3. Signal: python checkpoint.py signal assignment 021 "6 commissions ready" — 1 min
4. Done: Hand off to Managing Editor

Total work: ~1 hour (9:15-11:00 AM)
```

### **For Managing Editor (Write Issue 021)**
```
1. Scaffold: python scaffold.py issues/021 --sections 6 -- theme "[theme]" — 1 min
2. Write: 6 articles (250-300 words each) — 3-4 hours
3. Visuals: Create 6 ASCII visuals + cover + footer — 1-2 hours
4. Signal: python checkpoint.py signal managing 021 "All 6 sections complete..." — 1 min
5. Done: Hand off to Editorial Director

Total work: ~5 hours (12:00-6:45 PM)
```
