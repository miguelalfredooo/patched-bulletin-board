# Content Sources for Design By Bulletin

Curated publication catalog for Victor (Visual Curator), Maeve (Researcher), and Assignment Editor to scout signals, visual inspiration, and editorial content.

---

## 1. UX/UI & Product Design Publications

**Thought leadership, methodology, deep-dive case studies**

- **UX Collective** (Medium) — Essays on design systems, interaction design, research
- **Smashing Magazine** — In-depth tutorials, technique articles, design trends
- **Nielsen Norman Group (NN/g)** — Usability research, evidence-based design reports
- **Bootcamp** (Medium) — Product design case studies, design thinking methodology
- **A List Apart** — Web design and development craft, accessibility, standards
- **Prototypr.io** — Prototyping tools, design process, interaction patterns

**Why these matter:** Direct source for design methodology signals, emerging UX patterns, accessibility innovations.

---

## 2. Official Software & Company Blogs

**First-party announcements: new tool features, AI capabilities, structural changes**

- **Figma Blog / Figma Releases** — Design tool updates, API announcements, design system features
- **Adobe Newsroom & Adobe Blog** — Creative Cloud updates, acquisition announcements, AI tools (Firefly, etc.)
- **Webflow Blog** — Web design automation, no-code platforms, template releases
- **Google for Developers** — Material Design updates, AI capabilities (Gemini), Android design patterns
- **Apple Developer News** — iOS/macOS design guidelines, SwiftUI updates, design system announcements

**Why these matter:** Authoritative source for tool ecosystem changes, capability shifts, design system evolutions. First to know about emerging technical possibilities.

---

## 3. Major Tech & Business News

**Hardware rumors, industry acquisitions, legal/regulatory shifts, AI developments, community discussion**

- **TechCrunch** — Startup funding, product launches, AI developments, industry consolidation
- **The Verge** — Hardware design, tech criticism, design-focused product reviews
- **9to5Mac / MacRumors** — Apple design leaks, hardware rumors, ecosystem announcements
- **Fast Company** — Co.Design section specifically; business strategy, innovation trends
- **Wired** — Long-form tech journalism, culture intersections, design criticism
- **Hacker News** (news.ycombinator.com) — Community discussion on tech, tools, AI; designer/developer discourse; tool releases + critical commentary

**Why these matter:** Context for shifts in design practice (e.g., new iPhone form factor → design community responds), market consolidation (acquisitions change tool ecosystems), regulatory changes (copyright law → AI training debates). Hacker News specifically: early signals on emerging tools, practitioner reactions to announcements, contrarian perspectives on design trends.

---

## 4. Visual Inspiration & Graphic Design Magazines

**Typography, branding, illustration, aesthetic trends, visual culture**

- **It's Nice That** — Curated visual culture, emerging artists, design exhibitions, illustration trends
- **Creative Boom** — Design showcases, creative interviews, aesthetic trends, illustration
- **Behance** — Curated design projects, case studies, visual design portfolios (portfolio link source)
- **Print Mag** — Print design, typography, branding, visual identity systems
- **Awwwards** — Web design awards, interaction design showcases, digital aesthetic trends

**Why these matter:** Visual signals Victor needs. Emerging color palettes, typography directions, compositional patterns, aesthetic movements that inform design thinking 3–6 months ahead of mainstream adoption.

---

## 5. Independent Tool Makers & Repositories

**New tools, open-source systems, utility sites, independent design philosophy**

- **Product Hunt** — New design tool launches, design software releases, indie maker tools
- **GitHub** — Open-source design systems, CSS frameworks, design tokens, community tools
- **Standalone Utility Sites** — CSS generators, 3D UI libraries, AI photo editors, specialized tools
- **Substack / Independent Designer Blogs** — Personal essays, design philosophy, contrarian takes, practitioner insights

**Why these matter:** Emerging tool ecosystem, indie/open-source alternatives to major platforms, philosophical perspectives from design practitioners outside corporate environments.

## Scout Instructions for Curator & Researchers

### Maeve (Researcher) — Weekly Routine

1. **Monday–Wednesday:** Check official blogs and news sites
   - Figma Blog, Adobe Blog, Google for Developers, TechCrunch, The Verge
   - Note: New tools, feature launches, acquisition announcements
   
2. **Wednesday–Thursday:** Visual inspiration sites
   - It's Nice That, Creative Boom, Awwwards (daily new work)
   - Behance (weekly featured projects)
   - Identify: emerging trends, visual directions, aesthetic signals
   
3. **Thursday–Friday:** Deep-dive publications
   - Smashing Magazine, UX Collective, Nielsen Norman Group
   - A List Apart, Webflow Blog
   - Note: Case studies, methodology, research findings

4. **Ongoing:** Independent scouts
   - Product Hunt (new tools daily)
   - GitHub (new design system releases)
   - Substack (design philosophy, contrarian takes)

### Victor (Curator) — Visual Direction

**For each section, Victor scouts:**

1. What visual/aesthetic direction is emerging in the publications?
2. Are there color, typography, or compositional trends?
3. What quality threshold? (High/strong only — never mediocre)
4. Does this represent a distinct angle or category?

**Feeds to prioritize:** Awwwards, Behance, It's Nice That, Creative Boom (daily visual updates)

### Assignment Editor — Signal Finding

**After Victor's visual selections, Assignment Editor asks:**

1. What patterns emerge across the visuals?
2. What design thinking or philosophy is emerging?
3. What gaps exist in the coverage?
4. What stories could develop from these signals?

**Examples of signals from sources:**
- Figma Blog announces new AI features → Signal: "AI-assisted design becoming default expectation"
- It's Nice That features 5 typography-heavy projects → Signal: "Type as primary visual language"
- Nielsen Norman Group publishes accessibility research → Signal: "Accessibility as core design principle"

## Red Flags / Quality Filters

**Victor and Maeve should skip:**
- Generic listicles ("10 design trends for 2026")
- Promotional content (tool X launch without critical perspective)
- Outdated signals (6+ months old)
- Content with no clear design relevance
- Poor visual quality (even if conceptually interesting)

**Prioritize instead:**
- Original research (Nielsen Norman, UX studies)
- First-hand practitioner perspective (independent blogs, case studies)
- Visual work with distinct aesthetic point of view
- Critical analysis, not trend chasing
- Signals that are 2–4 weeks ahead of mainstream adoption
