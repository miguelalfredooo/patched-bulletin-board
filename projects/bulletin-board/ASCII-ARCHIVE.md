## Every piece, every issue

Append-only. One entry per issue.
Indexed by: issue, section, format, theme.

## How to read this archive

Each entry contains:
- The raw ASCII art (copy-paste ready for Telegram code blocks)
- Section, format type (A/B/C/D/E), and theme
- Character register (sparse / medium / dense / halftone)

**Format key:**
- A — Classic Object
- B — Geometric Frame
- C — Typographic
- D — Two Column
- E — Full Spread
