## Use Cases

**Compact-Left:**
- Alternating visual rhythm on page (ASCII вҖў Text вҖў ASCII вҖў Text)
- Left-to-right reading flow with ASCII as visual anchor
- Mobile layouts with limited width
- Social media card designs

**Compact-Right:**
- Mirror layout for alternating sections (Text вҖў ASCII вҖў Text вҖў ASCII)
- Right-to-left reading flow with ASCII as visual anchor
- Sidebar or column layouts
- Balanced asymmetric compositions

**Expanded:**
- Hero/feature display of single section
- Full-page visual preview
- Gallery or collection view
- Standalone ASCII art showcases
- Print layouts with generous spacing
