## Current Rules (ALL 5 VARIATIONS)
1. 11 pieces per issue (one per editorial section)
2. Pure text, UTF-8, monospace-compatible
3. Geometric Unicode style (в—Ҷ, в–І, в•‘, в•җ, etc.) only
4. Can be extracted and used independently

Expanded/Labeled:
- 14-line height with padding
- 80-character center-aligned width
- No additional context needed

Compact (Enhanced):
- 9-line height with editorial line
- ~60-character label column
- First line from Editorial Grid prose included
